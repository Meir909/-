# Happy Hearts Kindergarten

Премиальный сайт для детского сада на чистом HTML5/CSS3/Vanilla JS.

## Особенности
- 7 страниц (главная, о нас, воспитатели, галерея, новости, поступление, контакты)
- Трилингвальность: 🇰🇿 KZ / 🇷🇺 RU / 🇬🇧 EN — без перезагрузки, через localStorage
- Светлая и тёмная темы
- Glassmorphism, плавные анимации, scroll reveal
- Адаптивный дизайн (desktop / tablet / mobile)
- Floating WhatsApp кнопка с pulse-эффектом и динамическим сообщением
- Слайдер отзывов, FAQ accordion, lightbox для галереи
- Анимированные счётчики, мобильное меню, back-to-top
- Semantic HTML + meta/OG теги + favicon

## Структура

## Запуск
Откройте `public/index.html` в браузере или разместите содержимое `public/` на любом статическом хостинге.
